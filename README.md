# RECUIMA APK

**Calculadora Clínica RECUIMA** — Estratificación de Riesgo Cardiovascular

Escala de 7 variables con pesos específicos para predecir mortalidad intrahospitalaria por insuficiencia cardíaca.

## Instalación

1. Descarga el archivo `RECUIMA.apk` desde:  
   - 🌐 [Página oficial](https://adnova-design.github.io/RECUIMA-APK/)  
   - 📦 [Release v1.0.0](https://github.com/ADNova-Design/RECUIMA-APK/releases/download/v1.0.0/RECUIMA.apk) (descarga directa)
2. En tu dispositivo Android, habilita "Instalar apps de orígenes desconocidos"
3. Abre el archivo APK descargado e instala

## Uso

- Selecciona los criterios presentes (toggles) en el formulario
- La calculadora muestra el puntaje total y el nivel de riesgo automáticamente
- No requiere ningún dato personal del paciente

## Variables evaluadas

| Criterio | Peso |
|----------|------|
| Edad ≥ 70 años | 1 |
| Sexo femenino | 1 |
| HTA | 1 |
| FA (Fibrilación Auricular) | 1 |
| Killip IV | 1 |
| FV/TV (Fibrilación Ventricular / Taquicardia Ventricular) | 2 |
| FG < 60 mL/min | 3 |

**Puntaje ≥ 4** → Alto Riesgo
